구글 시트 연동
주소를 입력하면 학생 명단과 발송 기록이 구글 시트에 저장되어 모든 기기에서 공유됩니다. 비워두면 이 기기에만 저장됩니다. 인터넷이 끊겨도 마지막 명단은 캐시로 표시됩니다.
절약 모드
문자앱으로 보내기 (발송비 0원)
켜면 '발송' 버튼이 휴대폰 메시지 앱을 열고 내용·수신번호가 입력된 상태로 전환됩니다. 전송 버튼만 누르면 되고, 문자 무제한 요금제면 비용이 들지 않습니다. 단, 한 명씩만 보낼 수 있고 실제 전송 여부는 기록되지 않습니다(기록에는 '문자앱'으로 표시). 발신번호는 이 기기의 전화번호가 됩니다.
솔라피(Solapi) 연동
⚠️ 솔라피 설정이 필요합니다.
이 값들은 이 기기의 브라우저(localStorage)에만 저장되며, 선생님별로 각자의 발신번호를 입력해 본인 번호로 발송합니다. API Key/Secret과 발송 서버 주소는 공용 값을 입력하시면 됩니다.
발송 모드: 테스트 — 절약 모드가 켜져 있으면 문자앱으로, 꺼져 있고 위 4개 항목이 모두 채워지면 솔라피로 발송됩니다.
